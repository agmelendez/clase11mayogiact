1. Propósito analítico

En gestión marina costera se suelen observar muchas variables correlacionadas: presión turística, esfuerzo pesquero, calidad de agua, gobernanza local, conflictividad, infraestructura, cumplimiento normativo y percepción comunitaria. PCA permite reducir esa complejidad en dimensiones interpretables; el análisis de clúster permite construir tipologías de territorios, comunidades, playas, sitios de monitoreo o unidades pesqueras.

2. Reglas metodológicas mínimas

  1. Unidad de análisis clara: sitio, comunidad, playa, distrito, embarcación, hogar o actor institucional.
  2. Variables numéricas comparables: convertir escalas ordinales con cuidado y documentar la dirección de cada indicador.
  3. Estandarización obligatoria: usar z-score cuando las variables están en unidades distintas.
  4. Revisión de correlaciones: PCA es más útil cuando existe estructura de correlación; si todo es independiente, la reducción pierde sentido.
  5. Selección de componentes: combinar varianza acumulada, scree plot e interpretación sustantiva.
  6. Validación de clúster: comparar k con codo, silhouette y estabilidad conceptual.

Ruta de trabajo

Producto 1: matriz depurada y estandarizada.

Producto 3: clústeres interpretados y etiquetados.
Advertencia: los grupos estadísticos no son automáticamente grupos sociales o ecológicos reales. Deben validarse con conocimiento territorial.

# Laboratorio PCA + Clúster aplicado a gestión marina costera
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA
from sklearn.cluster import KMeans, AgglomerativeClustering
from sklearn.metrics import silhouette_score
from scipy.cluster.hierarchy import dendrogram, linkage

np.random.seed(123)
n = 180

df = pd.DataFrame({
    'presion_turistica': np.random.normal(60, 15, n),
    'esfuerzo_pesquero': np.random.normal(55, 18, n),
    'calidad_agua': np.random.normal(70, 12, n),
    'gobernanza_local': np.random.normal(50, 14, n),
    'cumplimiento_normativo': np.random.normal(58, 16, n),
    'conflictividad': np.random.normal(45, 17, n),
    'infraestructura': np.random.normal(62, 15, n),
    'biodiversidad': np.random.normal(65, 13, n)
})

# Ajustes para crear estructura correlacional realista
df['degradacion'] = 0.55*df['presion_turistica'] + 0.45*df['esfuerzo_pesquero'] - 0.35*df['calidad_agua'] + np.random.normal(0, 8, n)
df['capacidad_gestion'] = 0.50*df['gobernanza_local'] + 0.45*df['cumplimiento_normativo'] + 0.30*df['infraestructura'] + np.random.normal(0, 7, n)
variables = df.columns.tolist()

X = StandardScaler().fit_transform(df[variables])
pca = PCA().fit(X)
var_acum = np.cumsum(pca.explained_variance_ratio_)
n_comp = np.argmax(var_acum >= 0.75) + 1
pca_final = PCA(n_components=n_comp).fit(X)
scores = pca_final.transform(X)

sil = {}
for k in range(2, 8):
    labels = KMeans(n_clusters=k, n_init=30, random_state=123).fit_predict(scores)
    sil[k] = silhouette_score(scores, labels)
k_opt = max(sil, key=sil.get)
labels = KMeans(n_clusters=k_opt, n_init=50, random_state=123).fit_predict(scores)
df['cluster'] = labels

print('Componentes retenidos:', n_comp)
print('K óptimo por silhouette:', k_opt)
print(df.groupby('cluster')[variables].mean().round(2))
