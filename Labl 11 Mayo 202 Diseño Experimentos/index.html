Sesión 9: Diseño de Experimentos aplicado a Gestión Marina Costera

Laboratorio autoguiado para Google Colab. El eje ya no es ajustar una regresión observacional, sino planificar, simular, ejecutar y analizar un experimento factorial con réplicas, bloques, aleatorización, ANOVA, diagnóstico de supuestos e interpretación para decisiones de manejo costero.

🛠️ Instrumento de simulación DOE📋 Clasificador de diseño experimental

Teoría 1¿Por qué diseño de experimentos?

En gestión marina costera, muchas preguntas no se resuelven bien con datos puramente observacionales porque los sitios, las comunidades, las temporadas y las presiones ambientales pueden estar confundidas. El diseño de experimentos permite formular una intervención, controlar fuentes de variación, asignar tratamientos y estimar efectos con trazabilidad metodológica.

Pregunta central: ¿qué combinación de protección, restauración ecológica y control de presión turística produce mejores resultados en biodiversidad, calidad de hábitat o recuperación de biomasa?
ElementoTraducción experimentalEjemplo marino-costero
Unidad experimentalElemento al que se asigna un tratamientoTransecto, arrecife, parcela de manglar, playa monitoreada
FactorVariable manipulada o comparadaNivel de protección, restauración, presión turística
NivelCategoría de un factorBaja, media, alta protección
RespuestaIndicador medidoÍndice de biodiversidad, cobertura coralina, biomasa, turbidez

Teoría 2Factores, niveles y estructura factorial

Un diseño factorial estima efectos principales e interacciones. En un factorial 3 x 2 x 2 se evalúan tres niveles de protección, dos niveles de restauración y dos niveles de presión turística.

# Estructura conceptual del diseño
Respuesta = media general + efecto_proteccion + efecto_restauracion + efecto_turismo
            + interacciones + bloque/zona + error experimental
FactorNivelesHipótesis de gestión
ProtecciónBaja, Media, AltaMás protección debería mejorar biodiversidad o recuperación ecológica.
RestauraciónSin, ConLa restauración activa debería elevar la respuesta promedio.
Presión turísticaBaja, AltaMayor presión puede reducir biodiversidad o aumentar variabilidad.

Teoría 3Aleatorización, réplicas y bloques

La aleatorización protege contra sesgos de asignación; las réplicas permiten estimar error experimental; los bloques controlan variabilidad conocida entre zonas. En costa, los bloques son especialmente útiles porque una bahía, un estuario y un arrecife no son ambientalmente equivalentes.

Advertencia metodológica: si no hay réplicas, no se puede separar razonablemente el efecto del tratamiento del ruido local. Si no hay aleatorización, la inferencia causal queda debilitada.

Teoría 4ANOVA, GLM e interacciones

La salida principal del laboratorio es una tabla ANOVA para contrastar efectos principales e interacciones. Cuando la respuesta es continua y aproximadamente normal, ANOVA/OLS es suficiente. Si la respuesta es conteo, proporción o presencia/ausencia, se debe migrar a GLM o modelos mixtos.

RespuestaModelo recomendadoDiagnóstico clave
Índice continuo 0-100ANOVA / OLS factorialNormalidad residual, homocedasticidad, influencia
Conteo de especiesPoisson o binomial negativaSobredispersión
Proporción de coberturaBeta/GLM o transformaciónAcotamiento 0-1
Presencia/ausenciaLogísticaSeparación, calibración, odds ratios

Teoría 5Caso aplicado: restauración en zonas costeras

Suponga que una autoridad ambiental quiere evaluar si la restauración de hábitat mejora un índice de biodiversidad, pero sospecha que el resultado depende del nivel de protección y de la presión turística. Se diseña un experimento factorial con bloques por zona ecológica.

Decisión esperada: identificar si conviene priorizar restauración solo en zonas con protección alta, si el turismo reduce el efecto, o si se requieren medidas combinadas.

import itertools
import warnings
warnings.filterwarnings("ignore")

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import statsmodels.api as sm
import statsmodels.formula.api as smf
from scipy import stats
from statsmodels.stats.anova import anova_lm
from statsmodels.stats.multicomp import pairwise_tukeyhsd
from statsmodels.stats.diagnostic import het_breuschpagan

try:
    import seaborn as sns
    sns.set_theme(style="whitegrid")
except Exception:
    sns = None

# ------------------------------------------------------------
# 1. Simulación de un experimento factorial marino-costero
# ------------------------------------------------------------
def simular_experimento_marino(
    semilla=2026,
    replicas=5,
    incluir_bloques=True,
    escenario="base",
    error_sigma=4.5,
):
    """Genera un diseño factorial 3 x 2 x 2 con réplicas y bloques.

    Factores:
    - proteccion: Baja, Media, Alta
    - restauracion: Sin restauracion, Con restauracion
    - presion_turistica: Baja, Alta

    Respuesta:
    - biodiversidad: índice sintético 0-100 de biodiversidad/recuperación ecológica

    Escenarios:
    - base: supuestos razonables
    - interaccion_fuerte: efecto conjunto protección x restauración más alto
    - heterocedastico: varianza distinta por presión turística
    - outliers: incorpora observaciones influyentes
    - desbalanceado: elimina algunas celdas al azar para practicar diseños no balanceados
    """
    rng = np.random.default_rng(semilla)
    niveles = {
        "proteccion": ["Baja", "Media", "Alta"],
        "restauracion": ["Sin", "Con"],
        "presion_turistica": ["Baja", "Alta"],
    }
    bloques = ["Bahia", "Estuario", "Arrecife"] if incluir_bloques else ["General"]

    filas = []
    for bloque, prot, rest, turismo in itertools.product(
        bloques, niveles["proteccion"], niveles["restauracion"], niveles["presion_turistica"]
    ):
        for r in range(1, replicas + 1):
            filas.append({
                "bloque_zona": bloque,
                "proteccion": prot,
                "restauracion": rest,
                "presion_turistica": turismo,
                "replica": r,
            })
    df = pd.DataFrame(filas)
    df = df.sample(frac=1, random_state=semilla).reset_index(drop=True)
    df["orden_aleatorio"] = np.arange(1, len(df) + 1)

    # Efectos sustantivos simulados
    base = 52
    efecto_prot = {"Baja": -5, "Media": 4, "Alta": 11}
    efecto_rest = {"Sin": 0, "Con": 8}
    efecto_tur = {"Baja": 4, "Alta": -7}
    efecto_bloque = {"Bahia": -2, "Estuario": 1.5, "Arrecife": 4, "General": 0}

    eta = (
        base
        + df["proteccion"].map(efecto_prot)
        + df["restauracion"].map(efecto_rest)
        + df["presion_turistica"].map(efecto_tur)
        + df["bloque_zona"].map(efecto_bloque)
    ).astype(float)

    # Interacciones pedagógicas
    interaccion = (
        (df["proteccion"].eq("Alta") & df["restauracion"].eq("Con")) * 5
        + (df["proteccion"].eq("Baja") & df["presion_turistica"].eq("Alta")) * -4
    )
    if escenario == "interaccion_fuerte":
        interaccion = interaccion * 2.2
    eta = eta + interaccion

    if escenario == "heterocedastico":
        sigma = np.where(df["presion_turistica"].eq("Alta"), error_sigma * 1.9, error_sigma * 0.75)
    else:
        sigma = np.repeat(error_sigma, len(df))

    df["biodiversidad"] = eta + rng.normal(0, sigma, len(df))
    df["biodiversidad"] = df["biodiversidad"].clip(0, 100)

    if escenario == "outliers":
        idx = rng.choice(df.index, size=max(2, len(df)//20), replace=False)
        df.loc[idx, "biodiversidad"] = rng.choice([8, 96], size=len(idx))
        df.loc[idx, "observacion_atipica_simulada"] = 1
        df["observacion_atipica_simulada"] = df["observacion_atipica_simulada"].fillna(0).astype(int)

    if escenario == "desbalanceado":
        quitar = df.query("proteccion == 'Baja' and restauracion == 'Con' and presion_turistica == 'Alta'").sample(
            frac=0.45, random_state=semilla
        ).index
        df = df.drop(quitar).reset_index(drop=True)

    return df

Módulo 2Simulación de respuesta ecológica

La respuesta sintética se construye a partir de efectos principales, interacciones y error experimental. Esto permite practicar análisis aunque todavía no existan datos reales.

Parámetro sensible: el error sigma controla la variabilidad no explicada. Si aumenta, la potencia estadística baja y las diferencias reales son más difíciles de detectar.

# ------------------------------------------------------------
# 2. Ajuste del modelo experimental
# ------------------------------------------------------------
def ajustar_modelo_experimental(df, incluir_bloques=True):
    formula = "biodiversidad ~ C(proteccion) * C(restauracion) * C(presion_turistica)"
    if incluir_bloques and df["bloque_zona"].nunique() > 1:
        formula += " + C(bloque_zona)"
    modelo = smf.ols(formula, data=df).fit()
    return modelo, formula

# ------------------------------------------------------------
# 3. Diagnóstico estadístico del modelo
# ------------------------------------------------------------
def diagnosticar_modelo(modelo):
    resid = modelo.resid
    fitted = modelo.fittedvalues
    shapiro_stat, shapiro_p = stats.shapiro(resid)
    bp = het_breuschpagan(resid, modelo.model.exog)
    influence = modelo.get_influence()
    cooks = influence.cooks_distance[0]
    out = pd.DataFrame({
        "prueba": ["Shapiro-Wilk", "Breusch-Pagan", "Cook > 4/n"],
        "indicador": ["Normalidad residual", "Homocedasticidad", "Influencia"],
        "valor": [shapiro_p, bp[1], float(np.sum(cooks > 4/len(resid)))],
        "criterio": ["p > 0.05 deseable", "p > 0.05 deseable", "menor cantidad posible"],
    })
    return out, resid, fitted, cooks

# ------------------------------------------------------------
# 4. Visualizaciones clave
# ------------------------------------------------------------
def graficos_experimentales(df, modelo):
    resid = modelo.resid
    fitted = modelo.fittedvalues
    fig, axes = plt.subplots(2, 2, figsize=(14, 10))

    if sns:
        sns.pointplot(data=df, x="proteccion", y="biodiversidad", hue="restauracion", errorbar="se", ax=axes[0,0])
        sns.boxplot(data=df, x="presion_turistica", y="biodiversidad", hue="proteccion", ax=axes[0,1])
    else:
        df.groupby(["proteccion", "restauracion"])["biodiversidad"].mean().unstack().plot(ax=axes[0,0], marker="o")
        df.boxplot(column="biodiversidad", by="presion_turistica", ax=axes[0,1])

    axes[0,0].set_title("Interacción: protección x restauración")
    axes[0,1].set_title("Distribución por presión turística y protección")

    axes[1,0].scatter(fitted, resid, alpha=0.65)
    axes[1,0].axhline(0, linestyle="--")
    axes[1,0].set_title("Residuales vs valores ajustados")
    axes[1,0].set_xlabel("Valores ajustados")
    axes[1,0].set_ylabel("Residuales")

    sm.qqplot(resid, line="45", ax=axes[1,1])
    axes[1,1].set_title("Q-Q plot de residuales")
    plt.tight_layout()
    plt.show()
